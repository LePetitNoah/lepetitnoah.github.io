## Initialisation

> Installer Ruby [Lien](https://rubyinstaller.org/)

> Installer le bundler

```bash
gem install bundler
```

> Installer les gems

```bash
bundle install
```

> Lancer le serveur

```bash
bundle exec jekyll serve --livereload --trace
```
